## Status Legend

- **Draft**: Feature spec is being written
- **Planned**: Feature spec is complete, work not started
- **In Progress**: Implementation underway
- **In Review**: PR created, awaiting review
- **Done**: Feature shipped

## Creating a New Feature

1. Run `/feature-new <slug>` to create branch and spec
2. Run `/planner <slug>` to plan the implementation
3. Run `/execute <slug>` to implement
4. Run `/tests <slug>` to add tests
5. Run `/pr <slug>` to create PR
6. Run `/submit <slug>` to finalize
